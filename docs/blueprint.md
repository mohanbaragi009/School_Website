# **App Name**: Seshadripuram Gateway

## Core Features:

- Homepage Content Display: Render the primary sections from the school's existing website, including the news banner, main content area, principal messages, calendar, and academic video placeholders.
- Navigation System: Implement the hierarchical navigation menu (top header and footer links) as shown in the original screenshot, ensuring all links lead to their respective static content sections.
- Responsive Design: Ensure the layout and content adapt seamlessly across various device screen sizes, from mobile phones to desktop monitors, mirroring the visual fidelity of the provided image.
- Academic Calendar Interface: Display the monthly calendar view with event highlights as presented in the image, with placeholders for event details.
- AI Content Refinement Tool: Provide a simple text area tool where school staff can paste draft announcements or messages for the website and receive AI-generated suggestions for improvement in clarity, tone, or conciseness, enhancing the quality of published content.
- Contact Information Display: Clearly present the school's static contact details (address, phone, email) in the footer, matching the image's structure and providing direct click-to-contact functionality where appropriate.

## Style Guidelines:

- Primary color: Deep Prussian Blue (#264273), chosen to convey trust and professionalism befitting an educational institution, reflecting stability and academic rigor without resorting to literal or cliché representations.
- Background color: Light Cadet Gray (#ECF0F5), a heavily desaturated variant of the primary hue, providing a clean and serene canvas that maintains brand consistency while ensuring excellent readability for content.
- Accent color: Vibrant Medium Purple (#894DE0), chosen to be analogously distinct from the primary, offering a touch of creativity and vibrancy that draws attention to interactive elements and calls-to-action while complementing the primary scheme.
- Headline font: 'Roboto' (sans-serif) for a modern and accessible feel suitable for headers and key announcements. Note: currently only Google Fonts are supported.
- Body text font: 'Open Sans' (sans-serif) for its high readability and versatile application across all content sections, ensuring clarity for longer paragraphs. Note: currently only Google Fonts are supported.
- Utilize clean, outline-style vector icons that align with a professional and contemporary aesthetic, used for navigation helpers, social links, and interactive elements like 'More Information' buttons.
- Implement a hierarchical layout featuring a persistent header for navigation, distinct content blocks for news and messages arranged in a card-like structure, and a structured multi-column footer for organizational information.
- Incorporate subtle and swift hover effects on navigational links and interactive buttons to provide immediate visual feedback to the user, enhancing interface responsiveness without distracting from content consumption.
